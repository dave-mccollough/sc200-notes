# Microsoft Security and Zero Security

## Zero Trust

- Zero Trust is a Security Strategy
  - Not a product or service

- 3 core principals of Zero Trust
  - Trust Explicity
    - Always authenticate and authorize based on all available data points.
  - Use least privilege access
    - Limit user access with Just-In-Time and Just-Enough-Access (JIT/JEA), risk-based adaptive policies, and data protection.
  - Assume breach
    - Minimize blast radius and segment access. Verify end-to-end encryption and use analytics to get visibility, drive threat detection, and improve defenses.

- [Microsft Details](https://learn.microsoft.com/en-us/security/zero-trust/zero-trust-overview)
